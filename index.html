Boutique by DanstonCube

Plugin serveur pour la gestion des commerces sur DTC

Panneaux de ventes / échange / achat / donation / freebies pour des objets joueurs ou serveurs. Chaque panneau est reliable à un coffre ou stocker les objets à vendre (pour les objets des joueurs) mais également si l'on souhaite à une "vitrine" qui permet de visualiser l'objet à vendre. Attention, nécessite Mysql pour la gestions des transactions, Craftbukkit pour les apis, et iConomy5 pour la gestion des comptes en banque.

Dependencies

CrafBukkit - Bukkit API iConomy5 - Gestion des comptes en banque Mysql - Pour l'archivage des transactions effectuées.

Install

- Mettre le .jar dans le dossier "plugins" de bukkit - Lancer le serveur et l’arrêter une fois le chargement effectué. - Editer le fichier de configuration "plugins/Boutique/config.yml" et y corriger les identifiants de connexion à Mysql.
